What is Forged Gate?

Forged Gate is a high-performance API gateway built with a Rust data plane and Go control plane. It sits at the edge of your infrastructure, routing incoming requests to the appropriate backend services while providing authentication, rate limiting, and DDoS protection.

Unlike traditional API gateways that introduce latency and complexity, Forged Gate is designed for zero-overhead routing. The Rust data plane handles traffic with sub-millisecond latency, while the Go control plane provides a developer-friendly API for configuration and management.

Gate integrates natively with Forged Identity for authentication, Forged Govern for policy enforcement, and Forged Observe for telemetry—creating a complete edge layer for your platform.

Key Features

Multi-Upstream Routing

Route to any backend with path-based, header-based, or weighted routing. Support for HTTP, gRPC, and WebSocket protocols.

Authentication

Integrated with Forged Identity for OAuth2, JWT validation, and API key management. Supports multiple auth strategies per route.

Rate Limiting

Per-client, per-route rate limiting with sliding window algorithms. Configurable limits with automatic backpressure.

DDoS Protection

Automatic traffic analysis and mitigation at the edge. Real-time blocking of malicious requests with zero configuration.

TLS Termination

Automatic certificate management with ACME protocol support. SNI routing and HTTP/2 support out of the box.

Health Checks

Active and passive health monitoring with circuit breaker patterns. Automatic failover and graceful degradation.

Configuration

Configure Gate with simple YAML files or via the control plane API. Changes reload instantly with zero downtime. 

# gate.yaml
listeners:
  - port: 8000
    protocol: http
    tls:
      enabled: true
      auto_cert: true

routes:
  - name: api-v1
    path: /api/v1/*
    upstream: forged-erp:3000
    auth: required
    methods: [GET, POST, PUT, DELETE]

  - name: events
    path: /events/*
    upstream: forged-stream:9092
    rate_limit:
      requests: 1000
      window: 1m

  - name: public
    path: /public/*
    upstream: forged-cdn:80
    auth: none
    cache:
      enabled: true
      ttl: 1h

plugins:
  - cors:
      allow_origins: ["*"]
      allow_methods: ["GET", "POST"]

  - ddos_protection:
      enabled: true
      threshold: 10000

  - telemetry:
      exporter: forged-observe:4317
