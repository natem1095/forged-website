Overview

Unified Observability Platform

Forged Observe is a unified observability platform built with a Rust collector and Go control plane. It combines metrics, logs, and traces in a single system, replacing the Grafana/Prometheus/Loki/Tempo stack. Observe provides a simpler, more integrated alternative to Datadog, Grafana stack, Splunk, and New Relic with native OpenTelemetry support.

Configuration

Example Configuration

# observe.yaml
collector:
  otlp_grpc: 0.0.0.0:4317
  otlp_http: 0.0.0.0:4318
storage:
  metrics: /data/observe/metrics
  logs: /data/observe/logs
  traces: /data/observe/traces
retention:
  metrics: 30d
  logs: 14d
  traces: 7d
alerts:
  - name: high-latency
    query: histogram_quantile(0.99, rate(http_duration_seconds_bucket[5m])) > 0.5
    severity: warning
