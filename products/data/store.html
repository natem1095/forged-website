Overview

Full-Featured Relational Database

Built in Rust implementing the PostgreSQL wire protocol. Complete SQL execution engine, ACID transactions, buffer pool management, and catalog system. Drop-in replacement for PostgreSQL, MySQL, Amazon Aurora, and CockroachDB. Existing PostgreSQL drivers (psycopg2, pg, node-postgres, JDBC) connect unchanged — no code modifications required.

Drop-in Compatibility

Your Existing Code Works Unchanged

Standard PostgreSQL drivers connect directly to Forged Store. No application changes needed.

Python 
# Your existing code works unchanged
import psycopg2

conn = psycopg2.connect(
    host='localhost',
    port=5433,
    dbname='erp'
)

cur = conn.cursor()
cur.execute('SELECT * FROM orders WHERE status = %s', ('pending',))
results = cur.fetchall()
Node.js 
// Standard pg driver, just change the port
const { Pool } = require('pg');

const pool = new Pool({
  host: 'localhost',
  port: 5433,
  database: 'erp'
});

const result = await pool.query(
  'SELECT * FROM orders WHERE status = $1',
  ['pending']
);
Java 
// PostgreSQL JDBC driver works directly
String url = "jdbc:postgresql://localhost:5433/erp";
Connection conn = DriverManager.getConnection(url);

PreparedStatement stmt = conn.prepareStatement(
    "SELECT * FROM orders WHERE status = ?"
);
stmt.setString(1, "pending");
ResultSet rs = stmt.executeQuery();
Capabilities

Complete Database Engine

SQL Engine

Full SQL parsing, query planning, and execution. Supports complex joins, subqueries, aggregations, window functions, and CTEs.

ACID Transactions

Serializable isolation with multi-version concurrency control (MVCC). Full BEGIN/COMMIT/ROLLBACK support with savepoints.

Buffer Pool

Intelligent page caching with write-ahead logging (WAL). Configurable pool size with LRU eviction and dirty page management.

Catalog System

Full schema management and metadata storage. DDL support for CREATE/ALTER/DROP on tables, indexes, constraints, and views.

Index Support

B-tree and hash indexes for query acceleration. Automatic index selection by query planner with cost-based optimization.

Wire Protocol

Complete PostgreSQL v3 message protocol implementation. Binary and text formats, prepared statements, portal cursors, and COPY operations.
