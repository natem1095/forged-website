Purpose

Unified Platform Management

Forged Console is the single admin interface for the entire Forged platform. Monitor service health, manage configurations, provision users, and track all administrative actions from one unified dashboard.

Features

Core Capabilities

Service Health

Real-time status of all 18 services with alerts for downtime, performance degradation, and resource exhaustion.

Configuration Management

Centralized config editing with validation and rollback. Edit YAML/TOML configs with syntax highlighting and schema validation.

User Management

User provisioning, role assignment, access review, and multi-factor authentication enforcement across all services.

Audit Logs

Searchable log of all administrative actions with filtering by user, service, action type, and timestamp.

Resource Monitoring

CPU, memory, disk, network metrics per service with historical trends and capacity planning recommendations.

Deployment Management

Rolling updates, canary deployments, rollback with one-click deployment controls and deployment history tracking.

All Forged Services

Console connects to all 18 services via management APIs for configuration updates, health checks, and deployment control.
