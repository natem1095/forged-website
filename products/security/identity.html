Enterprise Identity & Access

Forged Identity delivers standards-based authentication and authorization with OAuth2/OIDC, hierarchical RBAC, and comprehensive session management. Replaces Okta, Auth0, Keycloak, and AWS Cognito with a unified, high-performance identity platform.

OAuth2 & OIDC

Full OAuth2 and OpenID Connect implementation with authorization code, client credentials, and device flow. Standards-compliant token issuance and validation.

RBAC

Role-based access control with hierarchical roles and permission inheritance. Fine-grained access policies with attribute-based extensions.

Single Sign-On

SAML 2.0 and OIDC federation for enterprise SSO integration. Support for external identity providers with automatic account provisioning.

Session Management

Distributed session storage with configurable lifetime and idle timeout. Cross-service session propagation with automatic renewal and revocation.

API Key Management

Service account provisioning with automatic key rotation and expiration. Scoped API keys with granular permission assignment and audit logging.

Multi-Factor Auth

TOTP and WebAuthn support for passwordless authentication. Hardware token integration with backup code generation and recovery flows.

Configuration

Identity Configuration

Configure Identity with OAuth2 providers, session policies, and authentication methods.

identity:
  port: 8300

  oauth2:
    issuer: https://identity.forged.local
    authorization_code:
      enabled: true
      pkce_required: true
    client_credentials:
      enabled: true
    device_flow:
      enabled: true
      poll_interval: 5s

  session:
    storage: cache
    lifetime: 8h
    idle_timeout: 30m
    renewal_window: 1h

  providers:
    saml:
      - name: enterprise_sso
        metadata_url: https://sso.corp.local/metadata
        auto_provision: true
    oidc:
      - name: google_workspace
        issuer: https://accounts.google.com
        client_id: ${GOOGLE_CLIENT_ID}

  mfa:
    totp:
      enabled: true
      issuer: Forged Identity
    webauthn:
      enabled: true
      rp_name: Forged Technologies
