Enterprise Secrets Management

Forged Vault provides zero-trust secrets management with military-grade encryption, automatic rotation, and comprehensive audit trails. Replaces HashiCorp Vault, AWS Secrets Manager, Azure Key Vault, and CyberArk with a unified, high-performance solution.

Encryption at Rest

AES-256-GCM encryption for all stored secrets with configurable key rotation policies. Supports multiple storage backends including disk, PostgreSQL, and distributed consensus.

Secret Versioning

KV v2 engine with automatic versioning and rollback capabilities. Maintain complete audit history of secret changes with configurable retention policies.

Dynamic Credentials

Generate short-lived database credentials, cloud provider tokens, and SSH keys on demand. Automatic credential rotation and lease management.

Shamir's Secret Sharing

Distributed unseal mechanism requiring M-of-N key shares to decrypt the master key. Prevents single point of compromise for vault initialization.

Audit Logging

Comprehensive audit trail of every secret access, modification, and administrative action. Tamper-proof logs with cryptographic verification.

Lease Management

Automatic expiration and renewal of dynamic secrets. Configurable TTL policies per secret type with grace period handling and revocation.

Configuration

Vault Configuration

Configure Vault with storage backend, seal mechanism, and audit logging.

vault:
  port: 8200
  storage:
    backend: raft
    path: /var/lib/vault/data
    node_id: vault-node-1

  seal:
    type: shamir
    shares: 5
    threshold: 3

  audit:
    file:
      path: /var/log/vault/audit.log
      format: json

  kv_v2:
    max_versions: 10
    delete_version_after: 90d

  dynamic:
    postgres:
      connection: "host=store port=5432"
      default_ttl: 1h
      max_ttl: 24h
